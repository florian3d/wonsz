# wonsz
Python SNAKE game clone. Written in Python 3.4 using PYGAME library.
<br/><br/>
W:  UP<br/>
S:  DOWN<br/>
A:  LEFT<br/>
D:  RIGHT<br/>
R:  PAUSE<br/>
N:  RESTART<br/>
X:  EXIT<br/>
1, 2, 3:  change speed
